# Madras Delights web

This is the codebase for [madrasdelights.com](https://madrasdelights.com/)

## How to use

Install dependencies and run:

```bash
# npm
npm install
npm run dev

# yarn
yarn
yarn dev

# pnpm
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
